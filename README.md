# Forecasting-ANTAMStockPrices
This study aims to find the best model for forecasting ANTM.JK stock prices from Jan 1, 2018, to Oct 30, 2023, by comparing ARIMAX-GARCH and LSTM methods using R-Studio and Python. The ARIMAX(2,1,2)-GARCH(1,1) model, incorporating world nickel prices as an exogenous variable, performed best with a MAPE of 7.206%.
